Tags can be configured in two ways: through the Discord commands or via the dashboard.

## Using the Dashboard

The [dashboard](https://dash.supportmail.dev/?next=tags) provides the most user-friendly experience for managing tags. It offers:

- **Visual Preview**: See how your tags will look before sending them
- **Intuitive UI**: A streamlined interface for creating, editing, and deleting tags
- **Better Organization**: Manage all your tags in one centralized location

## Using Discord Commands

Alternatively, you can manage tags directly in Discord using the `/config tags` command, which includes subcommands for:

- `/config tags add` **Adding Tags**: Create new tags with custom content
- `/config tags edit` **Editing Tags**: Modify existing tag content and properties
- `/config tags remove` **Removing Tags**: Delete tags you no longer need

## Recommendation

While both methods are available, we recommend using the dashboard for a more efficient workflow, especially when working with multiple tags or complex tag content.
The visual preview feature helps ensure your tags display correctly before deployment.
