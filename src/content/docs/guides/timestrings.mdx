### Usable time units

<div class="table-wrapper c1-right">

| Time Unit    | Usable key terms                       |
| ------------ | -------------------------------------- |
| Milliseconds | `ms, milli, millisecond, milliseconds` |
| Seconds      | `s, sec, secs, second, seconds`        |
| Minutes      | `m, min, mins, minute, minutes`        |
| Hours        | `h, hr, hrs, hour, hours`              |
| Days         | `d, day, days`                         |
| Weeks        | `w, week, weeks`                       |
| Months       | `mon, mth, mths, month, months`        |
| Years        | `y, yr, yrs, year, years`              |

</div>

### Usage

Just type it in.

`1 day` = **in 1 day**

`1h 30mins` = **in 1 hours and 30 minutes**

`2 hR 38sec` = **in 2 hours and 38 seconds**

`2h 30 minute` = **in 2 hours and 30 minutes**

<Aside type="tip">
It doesn't matter if you use upper or lower case, or if you include spaces.
</Aside>

### Nutzbare Zeiteinheiten

<div class="table-wrapper c1-right">

| Zeiteinheit   | Mögliche Schlüsselbegriffe               |
| ------------- | ---------------------------------------- |
| Millisekunden | `ms, milli, millisekunde, millisekunden` |
| Sekunden      | `s, sek, sekunde, sekunden`              |
| Minuten       | `m, min, mins, minute, minuten`          |
| Stunden       | `h, st, stund, stunden`                  |
| Tage          | `d, t, tag, tage`                        |
| Wochen        | `w, woche, wochen`                       |
| Monate        | `mon, mth, mons, monat, monate`          |
| Jahre         | `y, j, jh, jahr, jahre`                  |

</div>

### Nutzung

Einfach eintippen.

`1 tag` = **in 1 Tag**

`1st 30min` = **in 1 Stunde und 30 Minuten**

`23 mInUTeN 38sEc` = **in 23 Minuten and 38 Sekunden**

`2s 30 minute` = **in 2 Stunden and 30 Minuten**

<Aside type="tip">
Dabei spielt es keine Rolle, ob Groß- oder Kleinbuchstaben verwendet werden oder ob Leerzeichen enthalten sind.
</Aside>
